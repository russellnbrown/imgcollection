createimages.py is a python script to create some very simple test files.

Usage:   python createimages.py <location> <number of dirs> <number of files>

<location>        - the top directory containing all of the images
<number of dirs>  - the number of sub directories to create
<number of files> - the number of files per directory

The last file is written in different ways for search testing:
1. In a different format ( jpg or png )
2. An exact copy
3. With additional text added
